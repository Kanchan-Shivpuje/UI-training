Global Attributes:

Global attributes are attributes common to all HTML elements; they can be used on all elements, though the attributes may have no effect on some elements.

List of Global Attributes:

  • class - Specifies one or more class names for an element used in CSS.
  • id - Specifies a unique id for an element
  • style - Specifies an inline CSS style for an element.
  • title - Specifies extra information about an element
  • translate - Specifies whether the content of an element should be translated or not. etc are some global Attributes in HTML.

Element Specific Attributes:

These attributes are defined with the particular element.

List of Element Specific Attributes:

  • href- specifies link address in anchor tag which defines the links
  • src -specifies the file name of the image source in image tag
  • width - specifies the width of image, block, etc
  • height - specifies the height of image, block, etc
  • alt -specifies an alternative text to be used, if an image cannot be displayed.
